# Збір на дрони для ЗСУ × БФ «Алярм»

Особиста сторінка збору Євгена Мартинюка в межах відбору на Крайовий Вишкільний Табір (КВТ 30, Пласт НСОУ).

**Live:** https://evge14n.github.io/zbir-kvt30/

Збір — частина спільної цілі крайових пластових таборів і благодійного фонду «Алярм» (3 000 000 грн): автоматизація БпЛА для 80 ОДШБр, пікапи для ССО, модернізація 100 дронів для 59 ОШБр.

Статика без збірки: `index.html` + два зображення. Прогрес і посилання на банку оновлюються константами `GOAL` / `RAISED` / `BANKA_URL` у скрипті сторінки.
